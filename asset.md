# AI Image Asset Prompt Catalog

Use these prompts to generate square image sheets for the Spanish learning app. Recommended size is `1200x1200` because it stays below the requested maximum and splits cleanly into either a `4x4` grid of `300x300` cells or a `5x5` grid of `240x240` cells.

General cropping rules for every prompt:
- The output must be one square image sheet, not separate files.
- Draw visible grid boundaries only if your image tool needs them for alignment; otherwise keep cells visually separated by whitespace.
- Each cell must work as its own standalone picture after cropping.
- Do not include readable text, labels, letters, numbers, captions, watermarks, or UI elements.
- Keep the subject centered with clear margins on all sides.
- Use a consistent friendly educational illustration style across the whole sheet.
- Avoid heavy shadows, dark backgrounds, blurry subjects, and extreme closeups.
