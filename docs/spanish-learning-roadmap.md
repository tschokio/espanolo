# Spanish Learning Roadmap

This roadmap describes how Espanolo should support a learner starting with almost no Spanish and continuing into intermediate and advanced study. The app should not only add lessons; it should guide a repeatable learning loop: learn, retrieve, review, produce, and expand.

## Current Baseline

The project already has a strong A1 base:

- Guided lesson path with beginner grammar and vocabulary.
- Vocabulary decks with images, typing, recognition, and review.
- Spaced review for words, grammar exercises, and mistakes.
- Challenges, mini games, audio lab, progress, and admin tools.
- Roughly 49 seeded A1 lessons covering identity, ser/estar, articles, present verbs, food, travel, health, numbers, colors, weather, and simple survival phrases.

The next major goal is to make the system feel like a full learning journey instead of a collection of practice modes.

## Target Outcome

The long-term target is practical solo fluency support:

- A1: understand and produce basic personal, travel, food, location, and daily-life sentences.
- A2: handle common situations, talk about routines, needs, plans, likes, past events, and simple opinions.
- B1: explain experiences, tell stories, understand graded texts/audio, and hold structured conversations.
- B2 and beyond: maintain Spanish through reading, listening, writing, topic vocabulary, idioms, and advanced grammar.

The app cannot replace real conversation, but it can build the grammar, vocabulary, listening, and recall base needed for conversation to become useful.

## Daily Learning Loop

Every day should be guided by one clear flow:

1. Review due words, grammar, and mistakes.
2. Complete one lesson or one checkpoint.
3. Practice lesson vocabulary with active recall.
4. Produce Spanish: type, say, or build a full sentence.
5. Finish with a short challenge or mini game.

The dashboard should keep this loop visible and avoid scattering the learner across unrelated menu items.

## Weekly Learning Loop

Each week should include:

- One mixed checkpoint covering older material.
- One listening task with transcript reveal.
- One writing or speaking prompt.
- One targeted weak-area repair session.
- One optional challenge for speed and confidence.

## Phase 1: A1 Foundation Stabilization

Goal: make the current beginner path reliable, complete, and testable.

Curriculum focus:

- Pronouns, greetings, ser, estar, llamarse.
- Gender and articles: el, la, un, una.
- Plural articles and adjective agreement.
- Regular present tense: -ar, then -er and -ir.
- Core verbs: ser, estar, tener, querer, necesitar, ir, hacer, poder.
- Simple negation and question words.
- Location and basic prepositions.
- Travel, food, health, numbers, colors, weather, and repair phrases.

Product focus:

- Add unit structure to the course path.
- Add A1 checkpoints every 10-15 lessons.
- Add lesson-final production tasks.
- Track grammar patterns as weak spots, not only individual exercises.

Done when the learner can complete mixed A1 checkpoints without seeing answer choices first and can type short Spanish answers from English prompts.

## Phase 2: A1+ Daily-Life Control

Goal: move from isolated phrases to reusable daily-life sentence frames.

Curriculum focus:

- Daily routines and time expressions.
- Family and people descriptions.
- Shopping, prices, and restaurant conversations.
- Directions, transport, hotels, and common travel problems.
- Likes and dislikes with gustar, encantar, preferir.
- Near future: ir a + infinitive.
- Tener que, querer, poder, necesitar + infinitive.

Product focus:

- Add dialogue completion exercises.
- Add sentence transformation exercises.
- Add short answer prompts with expected model answers.
- Add daily-life scenario packs.

Done when the learner can handle basic travel, restaurant, shopping, and routine conversations in controlled practice.

## Phase 3: A2 Grammar Expansion

Goal: build enough grammar to talk about more than the present moment.

Curriculum focus:

- Irregular present tense: ir, hacer, decir, venir, salir, poner, traer.
- Reflexive verbs: levantarse, ducharse, llamarse.
- Direct objects: lo, la, los, las.
- Indirect objects: me, te, le, nos.
- Comparisons: mas que, menos que, tan como.
- Demonstratives: este, ese, aquel.
- Preterite tense intro.
- Imperfect tense intro.
- Present progressive: estoy hablando.
- Basic commands.

Product focus:

- Build a conjugation trainer by verb family.
- Add tense-specific review.
- Add translation both directions.
- Add "why was this wrong?" grammar explanations.

Done when the learner can describe current habits, future plans, basic past events, and simple preferences.

## Phase 4: A2 Real Conversation Blocks

Goal: teach complete conversation patterns instead of only grammar points.

Curriculum focus:

- Introduce yourself naturally.
- Explain a problem.
- Ask follow-up questions.
- Make and change plans.
- Talk about yesterday and tomorrow.
- Give simple opinions.
- Tell a short personal story.
- Repair conversation when lost.

Product focus:

- Add multi-turn dialogue practice.
- Add natural-response selection.
- Add speaking prompt cards with self-rating.
- Add timed recall rounds.
- Add scenario mastery badges.

Done when the learner can complete guided dialogues and produce short paragraphs or spoken answers from prompts.

## Phase 5: B1 Bridge

Goal: transition from controlled beginner Spanish to independent Spanish learning.

Curriculum focus:

- Preterite vs imperfect.
- Present perfect.
- Future and conditional.
- Subjunctive introduction.
- Por vs para.
- Se constructions.
- Relative clauses: que, donde, quien.
- Connectors: porque, aunque, entonces, sin embargo, por eso.
- Opinion and argument structures.

Product focus:

- Add graded reading lab.
- Add listening lab with transcript reveal and comprehension questions.
- Add writing mode with model answer comparison.
- Add mixed B1 weekly challenges.
- Add longer review sessions by skill: grammar, vocabulary, listening, writing.

Done when the learner can understand graded texts, answer comprehension questions, and explain opinions or experiences in simple Spanish.

## Phase 6: Advanced Maintenance

Goal: keep the app useful after the learner is already functional.

Curriculum focus:

- Topic vocabulary: work, finance, tech, relationships, travel, health, culture.
- Idioms and common native expressions.
- Verb nuance: quedar, llevar, seguir, echar, ponerse.
- Advanced subjunctive and conditional nuance.
- Debate, explanation, and storytelling.

Product focus:

- Add custom vocabulary import.
- Add article mode: read, answer, extract vocabulary.
- Add audio shadowing mode.
- Add monthly fluency tests.
- Add advanced writing prompts and correction rubrics.

Done when the app acts as a long-term Spanish maintenance system, not only a beginner course.
